# ViralQuiz

Mini-tests interactivos, visuales y virales construidos con Next.js.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Estructura

- `app/`: rutas de Next.js.
- `components/`: piezas reutilizables de UI.
- `data/quizzes.ts`: catalogo local de tests y preguntas.
- `public/images/`: recursos visuales servidos por la app.
